## Instalación

1. Clonar el repositorio:
```bash
git clone https://github.com/[usuario]/realestate-app.git
cd realestate-app
```

2. Instalar dependencias del frontend:
```bash
cd frontend
npm install
```

3. Configurar variables de entorno:
```bash
# Copiar el archivo de ejemplo y configurar
cp .env.example .env.local
```

## Uso

### Desarrollo

Para iniciar el servidor de desarrollo:

```bash
cd frontend
npm run dev
```

El aplicativo estará disponible en `http://localhost:3000`

### Producción

Para construir y ejecutar en producción:

```bash
cd frontend
npm run build
npm run start
```

## Scripts Disponibles

```bash
npm run dev      # Inicia el servidor de desarrollo
npm run build    # Construye la aplicación para producción
npm run start    # Inicia el servidor de producción
npm run lint     # Ejecuta el linter con Biome
npm run format   # Formatea el código con Biome
```
